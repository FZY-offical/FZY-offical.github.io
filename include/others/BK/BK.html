为什么我们叫BK Server / Why we are named 'BK Server'

我们起初的名字叫做FK Server
这是因为我们的成员经常会大吼'Fuck'
这十分文明——简直过于文明
叫习惯了 我们就取名为FK Server
但后来因为接触了'狗叫'的梗
我们的同学更喜欢用'Barking'来替代'Fucking'
而且我们接触的一款游戏'Keep Barking'(原:Keep talking ANE)
所以我们改名为BK Server
